# :school: Circlewha는
이화여자대학교에 소속된 동아리들과 이화여자대학교 학생들을 대상으로 하는 연합동아리에 관한 정보를 제공해드려요

## :school: 구현 기능
+ 소속/지원조건/필수지원학기/기타조건/관심분야에 따른 필터 및 검색을 이용하여 동아리를 찾을 수 있어요
+ 해당 동아리의 활동 시간/모집시기/활동 내용/활동 방식 등의 정보를 확인할 수 있어요

## :school: 직접 이용해보세요
